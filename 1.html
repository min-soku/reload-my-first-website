HTML이란 무엇인가?

HTML 은 웹 문서를 만들기 위하여 사용하는 기본적인 웹 언어의 한 종류이다. 하이퍼텍스트를 작성하기 위해 개발되었다.

// a태그는 하이퍼링크를 생성할 때 사용(새 창으로 열게 할 때 target="_blank"를 사용함), a태그에 title태그를 넣어 어디로 이동하는 지 작성함, img src=""에 넣고 싶은 사진의 이름을 적으면 사진이 삽입됨, width는 사진의 크기 조절

인터넷 서비스의 하나인 월드 와이드 웹을 통해 볼 수 있는 문서를 만들 때 사용하는 웹 언어의 한 종류이다. 특히 하이퍼텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 HTML로 작성된다.
